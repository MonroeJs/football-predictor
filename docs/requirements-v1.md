# 世界杯 2026 预测系统 — 需求方案 v1

## 一、概述

在现有 CLI + Flask Web 面板基础上，增加数据可视化和每日运营功能，提升用户体验和信息密度。

## 二、功能需求

### F1. 可视化增强

**目标**：图表代替纯文字，一眼看懂关键数据

**具体内容**：
- F1.1 置信度分布饼图/柱状图：今天/整体 VHigh/Elite/Max 各多少场
- F1.2 累计推荐投注金额柱状图：按天展示每天推荐总金额
- F1.3 分组热度图：展示12个小组各队实力对比（基于 Elo 或赔率）
- F1.4 置信度 vs 赔率散点图：看哪些比赛性价比高

**验收标准**：
- 页面加载即显示图表，无需额外点击
- 图表使用同一暗色主题（与现有 CSS 一致）
- 图表响应式，手机端也能看

### F2. 每日投注报告

**目标**：每天打开面板第一眼看到今天该投什么

**具体内容**：
- F2.1 今日比赛卡片：只显示当天日期的比赛
- F2.2 投注汇总：今天推荐投注场数、总金额、预期收益区间
- F2.3 按置信度排序：从 Max 到 High 递减
- F2.4 一键复制/导出推荐列表

**验收标准**：
- 默认首页视图切换到"今日"选项卡
- 投注汇总数据实时计算（基于赔率+置信度）

### F3. 赛程日历视图

**目标**：按日期浏览整个小组赛阶段的赛程

**具体内容**：
- F3.1 日历/时间线布局：6月11日至6月27日，每天一排
- F3.2 每天的比赛卡片：显示时间、对阵、置信度、推荐金额
- F3.3 快速跳转到今天

**验收标准**：
- 可以滚动浏览全部 17 个比赛日
- 点击某天跳转到当天比赛列表
- 已过去的日期显示灰色

### F4. 小组出线概率（扩展）

**目标**：看每个组各队出线概率

**具体内容**：
- F4.1 基于赔率计算每场胜率 → 模拟小组积分 → 出线概率
- F4.2 可视化：进度条或饼图展示出线概率
- F4.3 按组切换查看

**验收标准**：
- 概率总和为 100%（每组两个出线名额）
- 数据随赔率更新自动重算

## 三、非功能需求

- N1. 页面首屏加载 < 2s
- N2. 暗色主题，与现有设计一致
- N3. 图表在 Chrome/Edge 最新版正常显示
- N4. 所有数据计算在后端完成，前端只做展示

## 四、技术约束

- 前端：Bootstrap 5 + vanilla JS（不引入新框架）
- 图表：Chart.js 3.x（CDN 加载，轻量、暗色主题支持好）
- 后端：Flask + Jinja2，不新增 Python 依赖
- 数据：从现有 CSV + 预测 API 获取

## 五、交付物

1. `templates/wc_dashboard.html` — 更新后的主页面
2. `static/wc_charts.js` — 图表初始化脚本
3. `wc_app.py` — 新增路由和数据接口
4. `docs/development-v1.md` — 开发记录
