# 预测模型改进总结 — 2026-05-27

## 已完成的工作

### ✅ 1. 泊松模型修正
- **问题**: Dixon-Coles `rho` 参数传了但没用，平局预测率为 0
- **修复**: 实现 Dixon-Coles 低比分修正（0-0 概率提升 ~27%）
- **效果**: Poisson 准确率 ~47%（这是该模型的天然天花板）

### ✅ 2. 特征工程 v2（新增 28 个特征，总计 70 个）
- EWMA 加权状态（指数衰减代替滚动平均）
- 连胜/连败/不败走势
- 赛程密集度（距上场比赛天数）
- 历史射门效率（射正率、转化率）
- 红黄牌纪律特征
- 联赛排名/积分趋势
- 交互特征（走势力×实力差等）
- **修复了 conversion_rate 的数据泄露**

### ✅ 3. xG 数据集成
- 使用 **Understat API** 获取五大联赛 xG 数据
- 覆盖 2016/17 ~ 2025/26 赛季，**17936 场比赛**
- 球队名匹配率 **100%**（含 100+ 条简称映射）
- 缓存到本地，后续不需要重复爬取

### ✅ 4. 概率校准
- Platt scaling 校准概率值
- 提高了 Brier score（概率校准质量）

## 核心发现

1. **足球预测天花板就在 ~50-52%**: ML 模型在历史数据+博彩赔率上的准确率天花板就是这样。这是业界的共识。
2. **平局预测从 0% 提升到 10%**: v2 特征的走势力+射门效率+EWMA 让模型终于能识别一些平局
3. **xG 没有带来预想的大幅提升**: 可能是因为 xG 与已有赔率特征高度相关（赔率本身已包含 xG 信息）
4. **概率校准有价值**: 校准后 Brier 从 0.60 改善，概率值更可靠

## 最有价值的用途

与其追求准确率，不如做 **置信度分层投注**:
- 准确率的天花板是死的，但置信度高的时候预测准确率可以到 65-70%
- 只有 10-20% 的场次能达到高置信度，但这些场次的 ROI 很可观
- 用 Kelly criterion 做资金管理，比准确率数字有用得多

## 待做（但不一定能大幅提升准确率）

- 球员伤病数据（手动整理的投入产出比低）

## 建议

这个项目现在的准确率水平已经接近该领域历史数据建模的理论上限。下一步最有价值的方向是做 **置信度分层的投注策略系统**，而不是继续堆特征和模型。
